# Pand-A Day in My Life

A frontend-only React + Vite productivity companion app where a cozy panda reacts to goals, focus sessions, journal memories, streaks, rewards, outfits, decorations, and seasonal room changes.

## Tech Stack

- React + Vite
- Tailwind CSS
- dayjs
- localStorage
- No backend, database, or login system

## Features

- Home dashboard with panda companion, today's goals, XP, streaks, rewards, timer, and journal memory
- Monthly calendar with goals, notes, deadlines, difficulty, completion counts, and gentle missed-deadline messages
- Panda room with mood, level, XP, equipped outfit, decorations, and seasonal themes
- Daily journal that stores memories by date
- Daily reward system with one claim per day
- Achievements, unlockable outfits, and unlockable decorations
- Settings for timezone, notifications, theme, and local data reset

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The Vite config uses relative asset paths so the app can be deployed to GitHub Pages.
