# Pand-A Day in My Life

A cute panda-themed calendar productivity app for planning goals, keeping notes, and starting Pomodoro focus timers.

## Tech Stack

- React with Vite
- Tailwind CSS
- dayjs
- localStorage

## Features

- Monthly calendar view using local browser time
- Daily goals, notes, deadlines, completion tracking, and deletion
- Focus timer prompt after adding a goal
- Panda mascot mood states for idle, focusing, and celebrating
- Settings saved to localStorage
- Responsive pastel panda UI

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The Vite `base` value is configured for GitHub Pages-style static hosting.
